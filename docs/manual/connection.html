    Connection

    The Connection object is used internally to store the underlying Knex connection. It is accessible as database.connection.

    You probably don't need to know about this unless you're looking under the hood.

    Configuration

    The configuration options are the same as for Knex.js.

    Properties

    config

    A copy of the configuration options.

    knex

    A reference to the underlying Knex function/object.

    Methods

    raw(sql)

    A method to execute a raw query on the underlying Knex object.

    connection.raw(sql);

    This is equivalent to:

    connection.knex.raw(sql);

    pool()

    Returns the Knex client pool.

    Equivalent to:

    connection.knex.client.pool;

    acquire()

    Acquire a connection from the Knex client pool.

    Equivalent to:

    connection.knex.client.pool.acquire();

    destroy()

    Calls the destroy() method on the Knex pool.

    Functions

    connection()

    A function of convenience which wraps a call to new Connection().

    import { connection } from '@abw/badger-database';
const conn = connection(...);

    This is equivalent to:

    import { Connection } from '@abw/badger-database';
const conn = new Connection(...);
